# yxess

Yxess is a sexy, modular and blazing-fast static site generator based on Vanilla & Node JS.

## Installation
Ready to grab Yxess? Let's start! 

### From source (requires `npm`)
```
git clone https://github.com/yxess/yxess.git
cd yxess
npm install -g
```

### From `npm` packages
```bash
npm install -g yxess
```
